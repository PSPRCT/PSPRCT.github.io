Tips

  • Type near or near Dallas to jump the map and suggest nearby stores.
  • Use Within hubs to see only stores covered by visible hubs.
  • Pin hubs to compare coverage; Overlap highlights shared stores.
  • Side panel shows counts & metrics; click a store to zoom to it.
  • Use Views to save/restore your favorite setups.

Current Gap Finder

Current Gap Finder uses active/confirmed RTS coverage only. Future or pending hubs may appear on the map if toggled on, but they do not reduce current gap counts.

Manager rollups

Coverage Priorities by Manager

How to use: This ranks managers by current RTS coverage gap. Click a manager name to filter the map and open their Gap Finder summary.
Estimated RTS Needed: Count of consolidated 75-mile gap clusters with at least 25 uncovered stores. This is a planning estimate, not a staffing commitment.
Add 1 RTS Opportunity: Largest consolidated current gap cluster for that manager — a rough estimate of the best single placement opportunity.

Suggest RTS City

How to use: This ranks the best current RTS posting areas using active/confirmed RTS coverage only. It tests current uncovered clusters, searches nearby placement points, estimates how many currently uncovered stores one RTS could cover, and removes duplicate suggestions within the selected radius.
Important: This is a planning recommendation. It does not add a permanent RTS until you update your source JSON.

Weak-Pick / Swap Finder

This keeps the 100-trainer cap by identifying lower-value placements in the current Auto 100 plan and suggesting one-for-one replacements from remaining uncovered gap areas.

Leadership Summary

Final Plan Review Mode

Use this as the final placement meeting view. Mark each RTS placement as Approve, Maybe, or Replace, add notes, and export a leadership-ready review file while keeping the 100-trainer cap visible.

Compare Saved Plans

Save plan snapshots after each major version, then compare projected coverage, net-new stores, approved/replacement decisions, and placement differences.

State / Region Gap Summary

This estimates what remains uncovered after the active confirmed hubs plus the current proposed plan are applied. Existing confirmed Premium and Acosta hubs are counted as already filled/active before new proposed spots are added.

Replace-Marked Swap Recommendations

Mark placements as Replace in Final Plan Review Mode first. This protects Approved/Maybe/Pending rows and suggests one-for-one swaps only for those Replace-marked hubs, keeping the total 100-slot cap intact.
