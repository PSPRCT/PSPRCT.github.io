Help & Shortcuts

Legend filters: Click a legend item to toggle that coverage type on/off.

Nearest CSV: “Export Nearest CSV” lists each filtered store, its nearest visible hub, distance, and whether it’s within the current radius.

Saved views: Save your current filters + radius + toggles. Load later from the dropdown.

Clusters: Shift+click a cluster to see a coverage breakdown inside it.

Test hub: Turn on “Test hub”, then click the map to place (drag to move). The popup shows counts within the current radius.

Keyboard: R = reset · F = fit to results · H = toggle staffed hubs · O = toggle open hubs · C = toggle circles · / = focus search
