One Walmart Map Help

Fast workflow

  1. Start with Ops Intel → Leadership. Use it for the executive story: biggest gap, workload risk, dedicated overlay exposure, and next action.
  2. Use Coverage Priorities when you need manager-ranked gap ownership by RM or AM.
  3. Open a hub panel by clicking an RTS marker to review load, fringe stores, backup need, and stores in range.
  4. Use Dedicated Team Rollup to isolate Unilever, P&G, and Tyson without mixing them into core One Walmart filters.

What each tool is for

  • Current Gap Finder: finds concentrated uncovered store clusters using active RTS coverage only.
  • Manager Rollups: summarizes coverage and gaps by core One Walmart RM/AM hierarchy.
  • Coverage Priorities: ranks leaders by uncovered exposure and estimated staffing opportunity.
  • Suggest RTS City: planning-only candidate suggestions; it does not change permanent coverage until source JSON is updated.
  • Map Layers: heatmap, rings, overlap, compare, and stores-within-radius visibility controls.

Important guardrails

  • Active RTS hubs drive current coverage. Future RTS hubs are visual planning references only.
  • The One Walmart model improves network visibility and flexibility, but the routeable coverage rule remains 75 miles.
  • Dedicated teams are visibility overlays, not deductions from core One Walmart coverage.
  • Use store popups and RTS hub panels to validate decisions before making staffing recommendations.
